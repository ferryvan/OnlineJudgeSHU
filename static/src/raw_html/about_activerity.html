“盛大游戏杯”第15届上海大学程序设计联赛夏季赛暨上海高校金马五校赛 现场同步网络赛 活动相关

作者：lonelam     创建时间：2017年6月23日 23:43     最后更新：2017年6月24日 13:43


活动背景

2017年2月，华东师范大学、东华大学、华东理工大学、同济大学和上海大学等上海五所985和211高校的ACM教练决定在各自学校轮流举办ACM校际比赛，称为Shanghai STEED Programming Contest（上海金马五校程序设计竞赛，简称金马五校赛）。

ACM国际大学生程序设计竞赛（简称ACM/ICPC）是由国际计算机界历史最悠久、最具权威性的组织ACM（Association for Computing Machinery）主办，是世界上公认的规模最大、水平最高、参与人数最多的国际大学生程序设计竞赛，其目的旨在使大学生运用计算机来充分展示分析问题和解决问题的能力。该项竞赛自1970年至今，已成功举办40届，因历届竞赛都荟萃了世界各大洲的精英，云集了计算机界的“希望之星”，而受到国际各知名大学的重视，并受到全世界各著名计算机公司如Microsoft、IBM、Google、Facebook等公司的高度关注，成为世界各国大学生最具影响力的国际级计算机类的赛事，此项竞赛所颁发的获奖证书也为世界各著名计算机公司、各知名大学所认可。近年来国内越来越多的知名高校在计算机学科研究生复试中开始采用ACM试题作为上机考试内容。

盛大游戏

盛大游戏是全球领先的网络游戏开发商、运营商和发行商，拥有2000多名游戏研发、运维人员。盛大游戏有70多款网络游戏，承载超过18亿注册用户，致力于打造一个国际化的互动娱乐平台。

2001年，盛大游戏开始运营《热血传奇》，开创了中国的网络游戏时代。十余年间，盛大游戏一直扮演着中国游戏产业开创者和变革者的角色。到今年为止，盛大游戏旗下已拥有热血传奇，传奇世界，龙之谷，最终幻想14等著名大型游戏。

在移动领域，盛大游戏先后成功发行代理游戏《百万亚瑟王》与推出自研游戏《热血传奇手机版》，率先完成从端游为主到手游为重的企业战略转型。同时盛大游戏的IP战略布局初见成效：成功研发和发行首款国漫改编手游《血族》，知名IP《龙之谷》授权改编为动画电影，成为网游改编电影的先例。

目前盛大游戏在积极推进多个超级游戏IP的文学、动漫、影视、综艺等计划，构建一个以游戏为圆心的泛娱乐生态圈。

竞赛内容

上海大学将于2017年7月9日在宝山校区计算中心D楼举办上海金马五校程序设计竞赛上海大学站现场赛，最多可容纳600人参赛，参赛形式为个人赛，不收参赛费。在5个小时的比赛中，将有15道竞赛试题，其中5题的难度针对大一年级，6题的难度针对大二年级，4题的难度针对高年级。本次比赛旨在培养各高校大学生的程序设计能力，增进校际学术交流，在程序设计过程中激发创新意识，促进大学生程序设计的整体水平。详见http://acmoj.shu.edu.cn网站首页。

在上述现场赛比赛中，将同时举办现场同步网络赛，即在同一时间由同一组裁判评测同一套试题的程序代码。各个高校现在开始可以报名参加现场同步网络赛。

现场同步网络赛参赛对象

欢迎全国各个高校的本科生及研究生报名参加。
